# Development Phases

This directory contains resources, tools, and guides for each phase of the software development lifecycle.

## Workflow
1. **discovery**: Requirement analysis, market research, and feasibility studies.
2. **design**: UX/UI design, user flows, and prototyping.
3. **architecture**: System design, technology selection, and data modeling.
4. **implementation**: Coding, code review, and unit testing.
5. **testing**: Integration testing, system testing, and QA automation.
6. **deployment**: CI/CD, infrastructure as code, and release management.
7. **iteration**: Feedback loops, monitoring, and continuous improvement.

## How to Use
Navigate to the specific phase to find:
- **Prompts**: AI prompts tailored for this phase.
- **Tools**: Recommended AI tools.
- **Checklists**: Definition of Done (DoD) for the phase.
- **Handoffs**: What artifacts to produce for the next phase.
