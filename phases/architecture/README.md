# Architecture Phase

## Overview
Define the technical structure, data models, and system components.

## AI Tasks
- **System Design**: Suggest architectural patterns (e.g., microservices, serverless).
- **Data Modeling**: Generate SQL schemas or NoSQL document structures.
- **API Design**: Draft OpenAPI/Swagger specifications.
- **Tech Stack Selection**: Compare technologies based on requirements.

## Key Outputs
- System Architecture Diagram
- Database Schema
- API Contracts
- Technical Design Document (TDD)
