# Roles & Responsibilities

This directory contains guides and toolkits tailored for specific roles in the development team.

## Roles
- **Product Manager**: Market analysis, requirement gathering, roadmap planning.
- **UX/UI Designer**: User research, wireframing, prototyping, design systems.
- **Backend Engineer**: API design, database modeling, server-side logic.
- **Frontend Engineer**: UI implementation, state management, client-side logic.
- **Data Engineer**: Data pipelines, analytics, warehousing.
- **QA Engineer**: Test planning, automation, bug tracking.
- **DevOps Engineer**: Infrastructure, CI/CD, monitoring, security.
- **Project Manager**: Agile processes, resource management, timeline tracking.

## Cross-Functional Collaboration
AI tools can help bridge the gap between roles. For example, a Backend Engineer can use AI to generate API mocks for the Frontend Engineer, or a Product Manager can use AI to draft initial wireframes for the Designer.
