# 管理平台 UI 视觉基础规范文档（AI 生成执行版）V1.0

**适用平台**：平台框架、工作台、一级子页面

**规范用途**：指导AI生成页面、前端开发、UI设计

**遵循原则**：像素级、全局统一

**生成日期**：2026年03月

## 1 基础规范

### 1.1 布局体系

布局结构：B端三栏布局，包含顶部导航栏、左侧菜单栏、右侧内容区（内容区7:3划分，7为模块，3为待办审批、通知公告、系统消息）。

- **顶部导航栏**：通栏固定，高度60px
- **左侧菜单栏**：宽度250px、垂直可滚动、不可折叠
- **右侧内容区**：自适应宽度、可滚动
- **响应式**：PC端，内容区随浏览器宽度适配

### 1.2 字体规范

全局默认字体：Microsoft YaHei、PingFang SC、HarmonyOS_Sans

- 平台名称字号：20px
- 卡片标题字号：16px加粗
- 正文字号：14px默认
- 辅助字号：12px标签、时间、提示、注释
- 按钮字号：14px
- 菜单字号：14px默认
- 菜单选中字号：14px加粗
- 标签文字：12px

### 1.3 色彩体系

| 类型 | 色值 | 使用场景 |
|------|------|----------|
| 主色 | #3662EC | 按钮、选中、图标、高亮、边框、文字链接 |
| 辅助色9 | #3662EC；opacity：0.9 | 主按钮hover |
| 辅助色1 | #3662EC；opacity：0.1 | 菜单hover、标签背景、注释背景 |
| 成功色 | #22AC38 | 完成、通过、成功、已办结 |
| 警告色 | #FF6633 | 待处理、提醒、即将截止、注意 |
| 危险色 | #E4007F | 删除、退出、错误、驳回、禁止 |
| 边框色 | #D9D9D9 | 分割线、输入框、TAB下边线、表格外边线 |
| 背景色 | #3662EC；opacity：0.05 | 页面整体背景、内容区底色 |
| 白色 | #FFFFFF | 卡片、菜单、弹框背景 |
| 黑色 | #000000 | 正文、标题 |
| 描述色 | #666666 | 描述文字、注释 |
| 禁用色 | #CCCCCC | 禁用、停用、不可用 |

### 1.4 间距规范

**内容区间距：**

- 内容区内边距：10px
- 内容区卡片间距：10px，行间距：10px
- 内容区卡片内边距：20px，整体左对齐

**按钮间距：**

- 文字按钮：内边距：15px，最小宽度：70px，不限宽
- 文字按钮带下拉：内边距：15px，文字左对齐，下拉图标右对齐，图标居中，图标与文字水平居中间距15px，不限宽
- 图标+文字按钮：内边距：15px，图标居中，图标与文字水平居中间距：5px，不限宽
- 图标+文字按钮带下拉：内边距：15px，图标居中，图标与文字水平居中间距：5px，图标+文字左对齐，下拉图标右对齐，间距15px，不限宽

**菜单间距：**受菜单宽度影响，文字超长省略，整体左对齐

- 文字菜单：内边距：15px
- 图标+文字按钮：内边距：15px，图标居中，图标与文字水平居中间距：5px
- 图标+文字按钮带下拉：内边距：15px，图标居中，图标与文字水平居中间距：5px，图标+文字左对齐，下拉图标右对齐，间距15px

**文本行间距：**

- 文本行间距：20px
- 消息、通知、公告行间距：30px

### 1.5 圆角规范

- 卡片圆角：6px
- 按钮圆角：4px
- 输入框：4px
- 标签：半圆矩形
- 头像：正圆
- 弹框圆角：6px
- Banner圆角：6px

### 1.6 阴影规范

- 卡片默认：轻阴影（shadow-sm）
- 卡片悬浮：中等阴影，模拟上浮
- 弹框：重阴影，突出层级
- 顶部导航：无阴影
- 下拉菜单：轻阴影
- 按钮：无阴影

### 1.7 按钮规范

**主按钮**（重要操作：新增、提交、确认、审批）：

- 背景：主色，文字：白色，圆角：4px，内边距：15px，高度34px
- Hover：主色；opacity：0.9

**次按钮**（查询、编辑、查看、筛选）：

- 边框：1px 主色，文字：主色，背景：无
- Hover：主色；opacity：0.1

**危险按钮**（删除、驳回、退出）：

- 边框：1px 危险色，文字：危险色，背景：无
- Hover：危险色；opacity：0.1

**文字按钮**（查看更多、取消、忽略）：无背景无边框，文字主色

### 1.8 卡片规范

- 标题：短竖线4px*14px，圆角2px，主色 + 文字，字号：14px粗体
- 右侧可放：查看更多（文字按钮）
- 形态：圆角：6px，阴影：轻阴影，无边框
- 内边距：20px 20px
- 悬浮效果：上移2–3px，阴影加深，过渡0.2s平滑
- 卡片内禁止重复使用卡片

### 1.9 标签规范

- 字号：12px，内边距：3px 8px
- 待办/待处理：背景警告色，opacity：0.1，文字警告色
- 进行中/今日：背景主色，opacity：0.1，文字主色
- 已完成/通过：背景成功色，opacity：0.1，文字成功色
- 驳回/紧急：背景危险色，opacity：0.1，文字危险色

### 1.10 顶部导航栏

- 高度：60px，背景：主色，内边距：10px
- 左侧：校徽圆形40px + 学校名称 + 平台名称，文字20px
- 右侧：用户头像 + 姓名；下拉图标：12px

### 1.11 左侧菜单

- 宽度：250px，背景白色，右侧边框：1px solid 边框色
- 菜单栏内边距：10px
- 菜单项间距：10px
- 菜单项高度：40px，内边距：10px，圆角：4px
- 菜单图标宽度：18px*18px，图标居中，图标与文字水平居中，间距：10px

**菜单三态效果：**

- 默认：图标主色，文字黑色
- Hover：辅助色2，图标文字主色
- 选中Active：背景主色，图标白色，文字白色加粗

### 1.12 更多菜单

- 宽度：根据文字最多项给出宽度，顶部底部无内间距，向下叠加，菜单带阴影
- 菜单字号参照1.2字体规范
- 菜单项高度：40px
- 隔行用边框色opacity：0.5，1px分割

**菜单三态效果：**

- 默认：图标主色，文字黑色
- Hover：辅助色2，图标文字主色
- 选中Active：背景主色，图标白色，文字白色加粗

### 1.13 内容区规范

- 内容顺序：TAB选项卡（通栏） → 快捷操作 → 模块1 → 模块2 → 查询条件 → 表单
- 模块统一：卡片式，所有卡片统一居左、等宽排列，每一行最多3列

## 2 组件样式规范

### 2.1 弹框规范

- 宽度：480px / 600px / 1100px（三档固定）
- 背景白色，圆角：6px，标题：主色，高度40px，文字白色16px加粗左对齐，关闭按钮右对齐，内边距20px
- 底部按钮区：右对齐，次按钮在前，主按钮在后，内边距20px
- 遮罩层：rgba(0,0,0,0.3)，禁止点击空白处关闭

### 2.2 消息提示规范

- 成功提示：图标+文字成功色，内边距10px，半圆矩形，背景白色，轻阴影
- 错误提示：图标+文字危险色，内边距10px，半圆矩形，背景白色，轻阴影
- 警告提示：图标+文字警告色，内边距10px，半圆矩形，背景白色，轻阴影
- 显示位置：顶部居中，时长：10秒自动关闭

### 2.3 TAB选项卡规范

- 样式：高度40px，文字：14px黑色，无下边线
- 选中：文字主色，加粗，底部3px实线主色
- 间距：20px，切换无动画

### 2.4 表单规范

- 输入框高度：34px，圆角：4px，边框：1px 边框色
- 聚焦状态：边框主色
- 禁用状态：背景色，文字描述色
- 输入框内提示文字：14px描述色
- 校验错误：边框警告色，错误信息警告色，间距5px

### 2.5 表格规范

- 背景：白色，表头：背景：主色opacity：0.1，14px粗体
- 表格线：1px 边框色opacity：0.5，行高：40px
- 操作列：文字按钮

### 2.6 面包屑规范

- 位置：内容区最顶部，文字：14px描述色
- 当前页：主色，分隔符：/，间距：20px

### 2.7 控件内下拉菜单规范

- 菜单内边距：10px
- 菜单项高度：40px
- 文字按钮：默认主色，Hover：主色，opacity：0.1

### 2.8 搜索与查询规范

- 位置：内容区顶部
- 输入框：白色，圆角：6px，边距：10px
- 按钮：查询用主按钮，重置用次按钮
- 默认文字"请输入关键词搜索"，放大镜图标主色+文字描述色

### 2.9 注释规范

- 位置：内容区顶部，TAB选项下方
- 图标+文字：危险色
- 背景：危险色，opacity：0.1，内边距：20px，圆角：6px

### 2.10 查询栏规范

- 位置：内容区顶部，TAB选项下方
- 背景：白色，内边距：20px，圆角：6px
- 每个查询条件项：间距10px；查询按钮右对齐，底对齐

## 3 内容区规范

### 3.1 Banner区域规范

- 背景：渐变（主色 → opacity：0.7），文字颜色：白色
- 圆角：6px，内边距：20px–20px
- 布局：左侧文字区，右侧图标区
- 内容：欢迎语、今日待办、待审批、会议数、核心数据

### 3.2 列表规范

- 每行高度：50px，底部边框：1px #F3F4F6
- 布局：左侧主内容 + 右侧状态标签
- 勾选框：颜色#3662EC，文字超长显示省略号

### 3.3 数据统计页面

- 图表：主色，成功色，警告色，危险色

### 3.4 组织树结构规范

- 树结构宽度：250px，背景白色，右侧边框：1px solid 边框色
- 内边距10px，每行行间距：20px
- 一级左边距10px，二级左边距30px，三级左边距50px，四级左边距70px
- 文字左对齐，单行展示，超过内容省略，展开图标右对齐，图标与文字间距5px
- 级联复选或单选图标在文字左侧，图标与文字间距5px

**树结构搜索：**

- 位于树结构顶部
- 输入框：高度40px，圆角：6px，边距：10px，边框：1px solid 边框色
- 搜索图标+搜索提示文字间距5px整体左对齐，图标主色，提示文字描述色

**三态效果：**

- 默认：图标主色，文字黑色
- Hover：无变化
- 选中Active：文字主色加粗

## 4 交互规范

### 4.1 加载动画

- 位置：页面居中 / 模块居中
- 样式：环形旋转加载，主色
- 蒙层：半透明白色，800ms后自动消失

### 4.2 鼠标交互

- 所有可点击元素必须有Hover状态
- 过渡时间：0.2s–0.25s平滑
- 禁止突变、闪烁、生硬切换

### 4.3 表单交互规范

- 输入框聚焦：高亮边框
- 提交前：必填项校验，错误提示：红色文字
- 提交中：按钮禁用+加载状态

## 5 图标规范

### 5.1 图标

- 图标库：Font Awesome V5版
- 风格：线性/面性统一，禁止彩色卡通图标

## 6 特殊页面规范

### 6.1 空状态页面

- 居中图标，提示文字：14px描述色

### 6.2 404/异常页面

- 居中图标，提示文字：14px，描述色，提供返回按钮
